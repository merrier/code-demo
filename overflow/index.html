父元素随子元素宽度自动撑开问题, 父元素overflow: auto; 有滚动条时

父子元素分别设置宽度

                    .father {
                        width: 200px; height: 400px; border: 1px solid red; overflow: auto;
                    }
                    .child {
                        background: #ccc;
                    }
                    .child div {
                        width: 400px; height: 900px;
                    }

添加 min-width 和 display:inline-block 之后：

                    .container-2 .child {
                        min-width: 100%; display: inline-block;
                    }
